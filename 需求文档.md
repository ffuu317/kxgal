一、项目概述

1.1 项目名称

EatTruth（中文名：真食/信食）

1.2 一句话简介

基于AI水军检测和信用币经济的真实美食点评社区。

1.3 核心痛点

· 外卖/美食平台水军泛滥，虚假评论误导消费者
· 后厨照片“货不对板”，消费者无法验证
· 现有平台缺乏有效的用户激励机制

1.4 解决方案（四合一架构）

模块 作用
AI水军检测 实时识别虚假评论
个体性证明 指纹绑定，一人一账号
信用币经济 L0-L100等级系统
众包悬赏 用户到店拍照验证

1.5 目标用户

· 大学生：关注健康、预算有限的外卖主力
· 商家：诚信经营、希望被看到的小店
· 监督者：愿意为社区贡献的真实用户

四、数据库设计

4.1 表结构

```sql
-- 用户表
CREATE TABLE users (
    user_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    phone VARCHAR(11) UNIQUE NOT NULL,
    fingerprint_hash VARCHAR(255) NOT NULL,
    credit_balance INT DEFAULT 50,
    level INT DEFAULT 0,
    violation_count INT DEFAULT 0,
    status VARCHAR(20) DEFAULT 'active', -- active/banned/zombie
    avatar_url TEXT,
    id_card_verified BOOLEAN DEFAULT FALSE,
    last_login TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 指纹库表
CREATE TABLE fingerprints (
    fingerprint_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(user_id) ON DELETE CASCADE,
    finger_index VARCHAR(10) DEFAULT 'right_index',
    hash_value VARCHAR(255) NOT NULL,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 商家表
CREATE TABLE merchants (
    merchant_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(user_id),
    store_name VARCHAR(100) NOT NULL,
    latitude DECIMAL(10,8),
    longitude DECIMAL(11,8),
    address TEXT,
    license_url TEXT,
    last_photo_upload TIMESTAMP,
    status VARCHAR(20) DEFAULT 'pending', -- pending/approved/rejected
    created_at TIMESTAMP DEFAULT NOW()
);

-- 评论表
CREATE TABLE comments (
    comment_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(user_id),
    merchant_id UUID REFERENCES merchants(merchant_id),
    content TEXT NOT NULL,
    receipt_image TEXT, -- 消费截图
    likes INT DEFAULT 0,
    credit_cost INT DEFAULT 0,
    ai_spam_score DECIMAL(3,2), -- 0-1，越高越像水军
    status VARCHAR(20) DEFAULT 'pending', -- pending/published/spam
    created_at TIMESTAMP DEFAULT NOW()
);

-- 悬赏任务表
CREATE TABLE bounties (
    bounty_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    poster_id UUID REFERENCES users(user_id),
    merchant_id UUID REFERENCES merchants(merchant_id),
    reward_amount INT DEFAULT 50,
    latitude DECIMAL(10,8),
    longitude DECIMAL(11,8),
    status VARCHAR(20) DEFAULT 'pending', -- pending/accepted/completed
    created_at TIMESTAMP DEFAULT NOW()
);

-- 悬赏接单表
CREATE TABLE bounty_accepts (
    accept_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    bounty_id UUID REFERENCES bounties(bounty_id),
    accepter_id UUID REFERENCES users(user_id),
    photo_url TEXT,
    ai_check_result BOOLEAN,
    completed_at TIMESTAMP,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 信用币流水表
CREATE TABLE credit_transactions (
    tx_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(user_id),
    amount INT NOT NULL, -- 正数为收入，负数为支出
    balance_after INT,
    reason VARCHAR(50), -- login/comment/bounty/penalty/reward
    reference_id UUID, -- 关联评论ID或悬赏ID
    created_at TIMESTAMP DEFAULT NOW()
);

-- 商家超话圈-自传图
CREATE TABLE merchant_photos (
    photo_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    merchant_id UUID REFERENCES merchants(merchant_id),
    photo_url TEXT,
    month VARCHAR(7), -- YYYY-MM
    ai_verified BOOLEAN DEFAULT FALSE,
    created_at TIMESTAMP DEFAULT NOW()
);

-- 用户点赞记录
CREATE TABLE user_likes (
    like_id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    user_id UUID REFERENCES users(user_id),
    comment_id UUID REFERENCES comments(comment_id),
    created_at TIMESTAMP DEFAULT NOW(),
    UNIQUE(user_id, comment_id)
);
```

六、AI模块详细设计

6.1 水军评论检测

```python
# 使用Google Jigsaw模型
from transformers import pipeline

class SpamDetector:
    def __init__(self):
        self.classifier = pipeline(
            "text-classification",
            model="google/jigsaw-toxic-comment-classification"
        )
    
    def detect(self, text):
        result = self.classifier(text)[0]
        # 返回0-1之间的水军概率
        return result['score'] if result['label'] == 'toxic' else 1 - result['score']
```

6.2 图片重复检测

```python
import hashlib
from PIL import Image

def image_phash(image_path):
    # 计算感知哈希
    img = Image.open(image_path).resize((8,8)).convert('L')
    pixels = list(img.getdata())
    avg = sum(pixels) / len(pixels)
    hash_bits = ''.join('1' if p > avg else '0' for p in pixels)
    return hex(int(hash_bits, 2))

def is_duplicate(new_hash, existing_hashes, threshold=5):
    # 汉明距离比较
    for h in existing_hashes:
        if bin(int(new_hash,16) ^ int(h,16)).count('1') < threshold:
            return True
    return False
```

6.3 僵尸账号扫描

```python
from datetime import datetime, timedelta

def scan_zombie_accounts():
    cutoff_date = datetime.now() - timedelta(days=30)
    zombie_users = db.query(
        "SELECT user_id FROM users WHERE last_login < %s AND status = 'active'",
        [cutoff_date]
    )
    for user in zombie_users:
        db.execute(
            "UPDATE users SET status = 'zombie' WHERE user_id = %s",
            [user['user_id']]
        )
    return len(zombie_users)
```
